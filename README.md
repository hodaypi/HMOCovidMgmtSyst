* PyCharm 2019.2
* Firebase Realtime Database

## languages:
* Python
* HTML
* CSS
* JavaScript

## Packages:
to run this project, install it locally using pip:
* pip install flask
* pip install pyrebase

* There are 3 covid vaccine producers :
